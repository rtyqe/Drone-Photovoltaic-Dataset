# Drone-Photovoltaic-Dataset
This dataset is specifically designed for automatic detection of solar panel defects using drone aerial images and contains 1,468 high-resolution infrared images along with their corresponding YOLO format label files. These images show the condition of solar panels during operation and specifically annotate three types of defects.
